Diplômes et formations Développement web

Titre RNCP de Niveau 5: Développeur Intégrateur en réalisation d'applications web. (Equivalent bac+2)

Septembre 2020

Formation Développeur web front-end.

Formation Openclassrooms en alternance.

En cours (2 ans).

  • - Choisir une solution technique adaptée à votre client et travailler en mode Agile
  • - Créer des sites webs avec HTML et CSS
  • - Créer des applications web dynamiques avec Javascript et React
  • - Communiquer avec le back-end de l’application grâce à une API
  • - Mettre en œuvre des test unitaires et d’intégration et débugger le code
  • - Résoudre des problèmes techniques avec un langage de programmation et des algorithmes
  • - Refactoriser du code pour qu’il soit plus moderne et optimisé.
Formation Openclassrooms

Formations autodidactes validées sur le site Openclassrooms.

De septembre 2020 à février 2021

  • -Apprenez à créer votre site web avec HTML5 et CSS3
  • -Ecrivez du Javascript pour le web
  • -Apprenez à programmer avec Javascript
  • -Créez des pages web interactives avec JavaScript
  • -Adoptez les API REST pour vos projets web
  • -Concevez votre site web avec PHP et MySQL
  • -Apprenez à utiliser la ligne de commande dans un terminal
  • -Créez des sites web responsive avec Bootstrap 4
Formation Développeur Intégrateur web Junior

Formation intensive d'une durée de 6 mois, à la 3W Academy (Paris).

De Novembre 2019 à Juin 2020 (Validée)

    Formation aux langages :

  • - HTML et CSS pour l'intégration
  • - JavaScript pour l'intégration et le développement
  • - PHP pour la programmation
  • - SQL pour les bases de données
  • - Utilisation de Git pour travailler en groupe, collaborer et communiquer efficacement.
  • - Formation à l'utilisation du CMS Wordpress
